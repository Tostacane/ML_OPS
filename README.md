# Project Proposal – Employee Attrition Analysis

## Project Goal

The goal of this project is to analyze employee attrition and build a machine learning model to predict whether an employee is likely to leave the company.  
This is a supervised binary classification problem where the target variable is **Attrition (Yes/No)**.

## Dataset Source

Employee Attrition and Retention Analytics Dataset  
Source: Kaggle  
https://www.kaggle.com/datasets/ajinkyachintawar/employee-attrition-and-retention-analytics-dataset

It contains employee-level data with both numerical variables (e.g., Age, Monthly Income, Years at Company) and categorical variables (e.g., Job Role, Department, Overtime).

Each row represents one employee.

## Expected Outcome

The project aims to identify the most important factors influencing the decision of quitting from a firm, and evaluate our classification model.
